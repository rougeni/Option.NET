pour avoir accès au wre, il faut mettre l'adresse du dll dans le PATH.

pour cela, aller dans le panneau de configuration
	-> système
	-> sytème avancé
	-> variables d'environement
	-> dans les variables système, sélectionner PATH et cliquer sur modifier
	-> à la fin ajouter le chemin absolu vers le dll (dossier packages du projet)
